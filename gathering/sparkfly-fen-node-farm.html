Route:
Start at Orvar's Glen Waypoint [&BMQBAAA=]. This is a great farm to do while waiting for Tequatl. There is a permanent rich Platinum node at Verarium Delves (where the vista is [&BLUBAAA=]) and Shimmerstone Cave ([&BMwBAAA=]). The one in the cave is dangerous because there is a Champion Spider so beware. It could be best to skip it if you don't have a way to get it easily.

The herbs in this map are generally not worth it for the items, but decent for Volatile Magic with Volatile tools.
